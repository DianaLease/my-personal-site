## Diana Lease's Personal Site
This site is deployed at [https://www.dianalease.com/](https://www.dianalease.com/).

## To Run
```
gatsby build
gatsby serve
```

## To Develop
```
gatsby develop
```

## To Deploy

This site is automatically deployed from the master branch using Netlify.
