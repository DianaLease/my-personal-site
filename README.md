## Diana Lease's Personal Site
This site is deployed at [dianalease.com](www.dianalease.com).

## To Run
```
gatsby build
gatsby serve
```

## To Develop
```
gatsby develop
```

## To Deploy

```
gatsby build
```

## Built with Gatsby

This site was built using Gatsby. Full documentation for Gatsby lives [on the website](https://www.gatsbyjs.org/).
